# 임진왜란 트리거 및 이벤트 시나리오

## 1. 전체 구조

```
[게임 시작 Day 1]
       │
       ▼ 평화로운 조선기 (의뢰 1~23 진행 가능)
       │
[메인 의뢰 24 「강가의 이무기」]  ← 평판 60 + 4개 의뢰 완료 시 해금
       │
       ▼ 완료 즉시 컷씬
       │
[임진왜란 발발 — 임진 25년 4월]
       │
       ▼ 세계 상태 전환 (NPC 반응, 의뢰 게시판, 지역 접근 모두 변경)
       │
[피난민 호위 의뢰 반복 + 일본 용병 이벤트]
       │
       ▼ 평판 80 + 피난민 호위 5회 + 메인 24 완료
       │
[메인 25 「죽음의 행렬」 — 3단계 보스전]
       │
       ▼ 클리어
       │
[엔딩 분기 텍스트]
```

## 2. 트리거 조건 (정확히)

### 메인 24 해금 조건 (이무기 의뢰 등장)
- 게임 시간 ≥ Day 30
- 평판 ≥ 60
- 사이드 의뢰 4개 이상 완료
- 위 조건 충족 시 한양 의뢰 보드에 「조정의 부름」 표시

### 임진왜란 발발 조건
- **메인 24 「강가의 이무기」 완료 직후 무조건 발동**
- 시간 제한 없음 (플레이어 진행에 100% 의존)
- 단, 플레이어가 의뢰 안 받고 버티면 Day 90 시점에 「조정의 사신이 강제 호출」 이벤트 발생 (메인 24 강제 노출)

### 메인 25 해금 조건 (죽음의 행렬, 3개 의뢰로 분할)
- 임진왜란 발발 + 30일 경과
- 평판 ≥ 80
- 피난민 호위(13) 5회 이상 완료
- 모든 일본 용병 이벤트 1회 이상 진행
- 위 조건 충족 시 한양에 「만월 고개의 노승」 출현 → **25-A 의뢰 발생**
- 25-A 완료 → 25-B 자동 해금. 25-B 완료 → 25-C 자동 해금
- 각 단계 사이엔 자유 행동 가능 (다른 의뢰 진행, 회복, 영입 등)

> **모바일 페이싱**: 원래 3단계 연속 보스전 1개로 설계되었으나, 모바일 세션 길이(5–10분)에 맞춰 3개의 독립 의뢰로 분할. 각각 5–10분 분량

## 3. 트리거 컷씬 (이무기 완료 직후)

### 시나리오 「봉화의 밤」

**장면 1** — 한강 하류, 이무기 처치 직후
- 일행: 강가에 잔해. 이무기 비늘이 강물에 흩어짐
- 텍스트: 「큰 짐승의 숨이 멈췄다. 하늘이 무거워졌다」
- 정해은 대사: 「뭔가 잘못된 기분이… 약방의 약초들이 다 시들었다고 들었어요」
- 페이드 아웃

**장면 2** — 한양 부두 (귀환)
- 일행 부두 도착 → 부둣가가 평소와 다르게 인적 드뭄
- 거리에 사람들이 모여 봉화를 봄
- NPC 외침: 「왜놈들이… 동래에 상륙했다는 소식이오! 4월 13일, 부산이 함락됐소!」
- 일행 한 명(코타로 또는 료조)이 굳은 표정으로 멈춤

**장면 3** — 한양 동헌
- 한양 부윤이 일행을 부름
- 부윤: 「조정은 일행에게 보수를 두 배로 지불하고자 한다. 피난민을 호위하고 왜군 잔당과 결탁한 자들을 수색해주시오」
- 일행 (주인공 출신별 다른 대사):
  - 조선 출신: 「물론입니다. 이건 우리 일입니다」
  - 명 출신: 「명 조정에서도 곧 원군이 올 것이오」
  - 일본 출신: (잠시 침묵) 「내 칼이 이번엔 어디로 향해야 합니까」
- 부윤: 「자네들의 결정에 맡기겠소. 단, 지금부터 한양 거리는 자네들에게 호의적이지 않을 것이오」

**장면 4** — 화면 전환
- 큰 검은 화면 + 한자: **「壬辰倭亂」** + 한글 「임진왜란」
- 부제: 「임진 25년 4월 — 한반도 전역이 전화에 휩싸였다」
- BGM 변환: 평화 BGM → 위기 BGM (저음 북 + 단소)

## 4. 세계 상태 전환 (Post-Imjin)

### 4.1 NPC 반응

| 대상 | 변화 |
|------|------|
| 일반 NPC → 조선 출신 주인공 | 동정·격려 |
| 일반 NPC → 명 출신 주인공 | 우호 (원군의 동포) |
| 일반 NPC → 일본 출신 주인공 | **적대** (야유, 거래 거부, 일부 의뢰 받기 거부) |
| 시장 상인 → 일본 용병 | 가격 +20%, 일부 상인 거래 거부 |
| 도시 경비 → 일본 용병 | 임의 검문, 통과세 50냥 |

### 4.2 의뢰 보드 변화

**사라짐**:
- 「개경 사신 호송」(12) — 사신단이 본국으로 귀환
- 「옛 가야 유물 탐색」(15) — 지역 위험으로 폐쇄
- 「동래 포구의 떠돌이 검객」(3) — 동래 점령됨 (이미 코타로 영입 못 했다면 영구 손실)

**새로 등장**:
- 「피난민 호위」(13) — 무한 반복 의뢰
- 「왜군 잔당 추적」 절차생성 템플릿 T9 추가
- 「봉화 정찰」 절차생성 템플릿 T10 추가

**보수 변화**: 모든 의뢰 보수 +30% (위기 상황 프리미엄)

### 4.3 지역 접근

| 지역 | 변화 |
|------|------|
| 동래·부산 | 진입 불가 (왜군 점령) |
| 김해 | 진입 시 강제 전투 위험 (왜군 정찰) |
| 한양·개경·평양 | 정상 |
| 청송·태백·강원 산골 | 정상 (요괴 의뢰 계속) |
| 함흥 | 정상 |
| 제물포 | 진입 시 강제 전투 위험 |

지도 UI에 점령 지역은 붉은 색조로 표시.

### 4.4 시장·경제

- 식량 가격 +50%
- 무기·갑옷 가격 -20% (긴급 처분)
- 약초 가격 +30%
- 용병 영입비 +20% (수요 증가)

### 4.5 시간·BGM

- 한양 BGM 변경: 위기감 있는 어두운 톤
- 오버맵 BGM: 평소와 같지만 야간 인카운터 확률 +20%
- 시간 진행 가속: 휴식 비용 +50% (전쟁 중이라 자원 부족)

## 5. 일본 용병 이벤트 (Post-Imjin)

각 일본 용병에 1회성 분기 이벤트. 이벤트 트리거: 임진왜란 발발 후 해당 용병이 파티에 있고 한양 도착 시.

### 5.1 코타로 이벤트 「떠나야 하는가」

- 트리거: 임진왜란 발발 후 첫 한양 귀환, 코타로 파티에 포함
- 장면: 한양 숙소. 코타로가 주인공을 부름
- 대화:
  - 코타로: 「내 동포가 내 동료를 죽이고 있다. 나는 어찌해야 하나」
  - 선택:
    - **A: 「당신의 칼은 우리 편이다. 같이 가자」** → 충성도 +30, 영구 잔류 확정
    - **B: 「당신이 원하면 떠나도 좋다」** → 일시 이탈 (Day +30 후 자동 복귀, 「복귀의 결의」 패시브 강화)
    - **C: 「당신의 손으로 동포를 베어야 한다」** → 영구 이탈 (캐릭터 사라짐)
- 보상: A 선택 시 코타로의 패시브 「일도양단」 즉사 범위가 HP 30% → 35%로 강화

### 5.2 시노 이벤트 「가문의 명령」

- 트리거: 임진왜란 발발 후 첫 한양 귀환, 시노 파티에 포함
- 장면: 한양 야시장. 가문의 사자가 시노에게 비밀 임무 전달
- 대화:
  - 시노: 「내 가문에서 명을 보냈다. 너희 군 정보를 수집하라고. 거절할 수도 있다」
  - 선택:
    - **A: 「가문에 거짓 정보를 보내자」** → 충성도 +50, 영구 잔류, 「정보 수집」 능력 해금 (의뢰 사전 정보 -50% 비용)
    - **B: 「가문의 명대로 하라」** → 영구 이탈 (배신 처리, 미래 의뢰에서 적으로 등장 가능)
- 보상: A 선택 시 새 의뢰 해금 「가문의 진실」(시노 개인 사이드 의뢰, 보상 좋음)

### 5.3 료조 이벤트 「풍의 회상」

- 트리거: 임진왜란 발발 후 첫 한양 귀환, 료조 파티에 포함
- 장면: 한양 외곽. 료조가 활을 닦으며 회상
- 대화:
  - 료조: 「내 활이 사람을 쏘기 위한 것이었다는 걸 지금 깨닫는다. 무가의 운명이지」
  - 선택지 없음 (단순 회상)
- 보상: 료조의 패시브 「풍전사」가 1회/전투 → 2회/전투로 강화

### 5.4 한양 숙소 충돌 이벤트 (3국 갈등)

- 트리거: 임진왜란 발발 후, 일본 용병 + 조선 용병 둘 다 파티에 있고 한양 도착
- 장면: 숙소 안마당. 박돌쇠와 코타로(또는 료조)가 말다툼
- 대화:
  - 박돌쇠: 「네놈들이 다 같은 핏줄이지, 누구는 왜군 누구는 우리 편이 어딨어」
  - 코타로: 「난 내 칼을 너희 편에 두었다. 그 이상 뭘 더 원하나」
- 선택:
  - **A: 「우린 같은 용병단이다. 그만하자」** → 양측 충성도 +10
  - **B: 「출신은 출신이지. 박돌쇠 말도 일리 있다」** → 일본 용병 충성도 -20
  - **C: 「과거가 어떻든 칼끝의 방향이 중요하다」** → 코타로 충성도 +20, 박돌쇠 충성도 -5

각 선택은 메인 25 클리어 시 엔딩 텍스트에 반영.

## 6. 메인 25 「죽음의 행렬」 시나리오 (3개 의뢰 분할)

원래 3단계 보스전 1회로 설계했으나 모바일 페이싱(5–10분 세션)에 맞춰 **3개 독립 의뢰**로 분할. 각 단계 사이 자유 행동 가능.

### 사전 빌드업
- 한양에 「만월 고개의 노승」 출현
- 노승: 「부패한 자들이 저승의 문을 강제로 열고 있다. 한양이 위태롭다」
- 의뢰 보드에 25-A 노출

### 25-A 「그림자 결계」 (단일 전투, 5–7분)
- 위치: 만월 고개 입구
- 적: 그림자 손각시 3 (영체 처치 → 캐스터 필수, 정해은 또는 왕청림 출전 강제 권장)
- 클리어 시: 노승의 단서 → 25-B 해금
- 줄거리: 결계가 부서지자 음모자가 도주. 추적 단서 획득

### 25-B 「의식 제단」 (단일 전투, 5–7분)
- 위치: 만월 고개 중턱
- 적: 음모자(인간) 4 (부패 관료 + 일본 첩자 혼합)
- **분기 선택**: 전투 시작 전 「협상 시도」 옵션. 성공(평판 80↑) 시 1명이 이탈해 다음 단계에서 저승사자 디버프 정보 제공
- 클리어 시: 25-C 해금

### 25-C 「강림한 사자」 (보스전, 7–10분)
- 위치: 만월 고개 정상
- 적: 저승사자 1 (보스 메커닉 풀가동: 명부 시스템, 혼 거두기, 저승의 한기)
- 25-B에서 협상 성공 시: 저승사자 「명부 기록」 주기 +1턴 (난이도 ↓)
- 정해은 「축귀침」 + 왕청림 「귀살부」가 사실상 필수
- **광고 옵션**: 의뢰 시작 전 보상 광고 시청 시 「부적 1회 무료」 (즉사 1회 무효)

### 25-C 클리어 직후 컷씬

- 저승사자 사라짐. 명부에서 일행의 이름이 지워짐
- 노승: 「잠시의 평화일 뿐이오. 전쟁은 끝나지 않았소. 하지만 자네들은 한양을 지켰소」
- 한양 부윤: 「조선의 이름으로 자네들에게 감사하오」
- 화면 전환: 엔딩 텍스트

### 엔딩 텍스트 분기

분기 조건:
- 주인공 출신 (조선/명/일본)
- 잔류한 일본 용병 수 (3/2/1/0)
- 충돌 이벤트 선택 결과

엔딩은 짧은 한 화면 텍스트 (3~5문장) + 일행 일러스트. 다음 분기 정도면 충분:

| 주인공 출신 | 일본 잔류 ≥2 | 일본 잔류 ≤1 |
|------------|--------------|--------------|
| 조선 | 「한 깃발 아래」 — 우호 엔딩 | 「갈라진 길」 — 쓴맛 |
| 명 | 「두 제국의 사이」 — 외교 엔딩 | 「귀향의 결심」 |
| 일본 | 「조국을 등진 칼」 — 추방 엔딩 | 「혼자가 된 검」 |

MVP 엔딩은 텍스트만. 일러스트는 post-launch 보강.

## 8. 디자인 원칙 (재확인)

1. **트리거는 플레이어 행동 기반** (메인 24 완료). 시간 강제 트리거는 백업
2. **세계 변화는 시각/청각/시스템 모두 반영**. BGM + UI 컬러 + 의뢰 보드 + NPC 대사
