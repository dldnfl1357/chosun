# 의뢰 25개 세부 설계 (MVP, 모바일)

## 모바일 페이싱 규칙

- **기본 세션 길이**: 의뢰당 5–10분 = 의뢰 수락(10s) + 오버맵 이동(15s) + 전투(3–5분) + 보상 화면(20s) + 인터스티셜(조건부 5–15s)
- **단일 전투 의뢰가 기본**. 다단계 전투 의뢰는 「★ 다단계」로 표기, 15–20분 소요
- **적 수 가이드**: 5명 파티 기준 잡몹 4~6, 중급 단일/소수 2~3, 보스 1+조무사 2~3
- **보상 광고 통합**:
  - 「보수 ×2」 — 의뢰 보수 두 배 (보상 광고)
  - 「무료 회복」 — 다운된 캐릭터 1명 즉시 부활 (전투 패배 후)
  - 「급행 회복」 — 부상 회복 기간 50% 단축 (보상 광고)
- **인터스티셜**: 의뢰 종료 3회마다 1회. 광고 제거 IAP 보유 시 미발동

## 1. 청송 산적단 소탕 (입문)

- **위치**: 청송 산간 노드
- **의뢰주**: 청송 현감
- **목표**: 행인을 노리는 산적단 처치
- **적**: 산적(인간) 3 + 들도깨비 2
- **평판**: 0 / **보수**: 100냥 / **보상**: 평판 +10
- **줄거리**: 한양 용병 회관 첫 의뢰. NPC가 「변경엔 사람도 도깨비도 도적이지」라는 톤을 깔아줌
- **튜토리얼 의뢰**: 그리드 이동·평타·일시정지 강제 안내

## 2. 개성 약재상의 의뢰 (정해은 영입)

- **위치**: 개성 시장
- **의뢰주**: 약재상 「정 영감」
- **목표**: 산속 약초 채집 중 사라진 견습 의원(정해은) 찾기 → 폐가에서 달걀귀신과 싸우는 그녀 구원
- **적**: 달걀귀신 4
- **평판**: 20 / **보수**: 150냥 / **보상**: 정해은 영입 가능 해금
- **줄거리**: 정해은이 누이를 죽인 요괴를 추적하다 발이 묶임. 의뢰 완료 후 그녀가 합류 제안

## 3. 동래 포구의 떠돌이 검객 (코타로 영입)

- **위치**: 동래 부두
- **의뢰주**: 동래 부사
- **목표**: 부두에서 행패 부리는 일본 낭인 진압. 도착해보니 그는 자기 주군을 죽인 자를 찾고 있으며, 결투를 청함
- **적**: 코타로(NPC 결투) 1 + 그의 적 일본 무리 3 (전투 중 합세)
- **평판**: 25 / **보수**: 200냥 / **보상**: 코타로 영입 가능 해금
- **줄거리**: 결투 1대1로 시작 → 진짜 적이 난입 → 코타로와 함께 싸우게 됨. 끝나면 합류 제안

## 4. 임진강변의 흑궁수 (료조 영입)

- **위치**: 임진강 나루
- **의뢰주**: 강변 마을 촌장
- **목표**: 강변에 출몰하는 「검은 옷의 활잡이」 정체 확인 → 료조임 발각 → 료조와 함께 인근 도적 처치
- **적**: 도적 5 (활잡이 2 포함)
- **평판**: 30 / **보수**: 250냥 / **보상**: 료조 영입 가능 해금
- **줄거리**: 료조가 의식만으로 만족 못해 실전 도적 사냥 중. 「내 활을 시험할 동료가 필요하다」

## 5. 제물포 산 자를 부르는 자 (왕청림 영입)

- **위치**: 제물포 항구
- **의뢰주**: 무역상 「최 행수」
- **목표**: 항구에 출몰하는 영물 정체 파악 → 손각시 처치
- **적**: 그림자 손각시 3
- **평판**: 40 / **보수**: 300냥 / **보상**: 왕청림 영입 가능 해금 (그가 도와주러 등장)
- **줄거리**: 의뢰 진행 중 왕청림 등장, 함께 손각시 정리. 일행의 「영체 대응 능력 부족」을 그가 지적하며 합류

- **평판**: 50 / **보수**: 400냥 / **보상**: 시노 영입 가능
- **줄거리**: 시노가 적대 가문의 음모를 막던 중. 「내 가문은 죽었다. 너희 가문이 필요하다」

## 24. [메인] 강가의 이무기 ★ 다단계 (전초전 + 보스)

- **위치**: 한강 하류
- **의뢰주**: 조정 (직접 호출)
- **목표**: 한강에 나타난 이무기 처치. 사전 의뢰 4개 이상 완료 + 평판 60 이상 시 호출
- **전투 1 (전초)**: 들도깨비 4 + 잡몹 시중
- **전투 2 (보스)**: 이무기 1 (3개 부위 + 보스 메커닉) + 들도깨비 2
- **평판**: 60 (조건) / **보수**: 1000냥 / **보상**: 이무기 비늘 (방어구 대거), 평판 +50, **임진왜란 발발 트리거**
- **다단계 구조**: 전투1 ↔ 전투2 사이 회복 + 자동저장. 보스 도전 전에 보상 광고로 「전초전 보상 2배」 옵션
- **줄거리**: 의뢰 완료 → 컷씬 → 동래에 왜군 상륙 보고 → 임진왜란 발발. 한양에 비상령

- **의뢰주**: 만월 고개의 노승
- **적**: 그림자 손각시 3 (영체 처치 = 캐스터 필수)
- **평판**: 80 + 피난민 호위 5회 + 메인 24 완료 / **보수**: 600냥
- **줄거리**: 노승 등장, 음모 단서. 클리어 후 「의식 제단」 의뢰 해금

### 25-B. 의식 제단 (단일 전투, 5–7분)
- **위치**: 만월 고개 중턱
- **의뢰주**: 25-A 완료 후 자동
- **적**: 음모자(인간) 4 (부패 관료 + 일본 첩자)
- **보수**: 600냥
- **분기 선택**: 음모자 중 1명은 협상 가능 (대화 옵션). 협상 성공 시 25-C에서 보스 디버프
- **클리어 후**: 「강림한 사자」 의뢰 해금

### 25-C. 강림한 사자 (보스 단일 전투, 7–10분)
- **위치**: 만월 고개 정상
- **의뢰주**: 25-B 완료 후 자동
- **적**: 저승사자 1 (보스 메커닉 풀가동)
- **보수**: 1500냥 / **보상**: 명부 조각 (엔딩 트리거), 평판 +100
- **광고 옵션**: 의뢰 시작 전 보상 광고 시청 시 「부적 1회 무료」 제공
- **클리어**: 엔딩 컷씬

## 의뢰 시스템 데이터 모델 (구현용)

```csharp
[CreateAssetMenu]
class ContractDef : ScriptableObject {
  string Id;
  string DisplayName;
  ContractType Type;            // Kill | Escort | Search | Main | Generated
  string LocationNodeId;
  int RepRequired;
  int RewardGold;
  int RewardRep;
  ItemDef[] RewardItems;
  EnemyComposition[] Encounters;  // 다회 전투 의뢰 지원
  string[] PrerequisiteContractIds;
  bool RequiresImjin;             // post-imjin 전용
  bool BlockedByImjin;            // imjin 후 사라짐
  DialogueScript IntroDialog;
  DialogueScript OutroDialog;
  RecruitDef UnlockedRecruit;     // 영입 트리거용
}
```

절차생성 의뢰는 `GeneratedContractTemplate`로 별도 정의, 런타임에 파라미터 채워서 `ContractInstance` 생성.

## 디자인 원칙 (재확인)

1. **첫 5시간 = 평판 0~40 의뢰 ~10개로 채워짐**. 입문 → 영입 → 본격 사이클
2. **고유 의뢰는 영입과 묶음**. 5/9 용병이 의뢰로 영입 (강한 모티베이션)
3. **절차생성은 보수·평판 채우기용**. 콘텐츠 인플레이션 없이 반복 가능
4. **메인 24가 임진왜란 트리거**. 시간 + 평판 조건이 충족된 시점 발동
5. **피난민 호위(13)는 post-imjin 무한 의뢰**. 후반부 게임플레이 유지
