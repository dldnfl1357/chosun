# 용병 9명 세부 설계 (MVP, 모바일)

3국 × 3직업 = 9명. 역할 분배는 **탱커 2 / 근접 DPS 2 / 원거리 DPS 2 / 힐러 1 / 캐스터 1 / 버스트 1**로 균형.

**모바일 파티 슬롯**: 주인공 1 + 용병 최대 4 = 출전 5명 (전체 명단 10명에서 의뢰 전 선발). 한 의뢰에 모든 용병을 동시에 데려갈 수 없으므로 의뢰 성격에 맞춘 편성이 게임플레이의 큰 부분.

## 역할 분포 한눈에

| 역할 | 캐릭터 |
|------|--------|
| 방어형 탱커 | 김덕림(조선·무관) |
| 반격형 탱커 | 호중산(명·표사) |
| 근접 DPS (속공·연타) | 진소걸(명·권법가) |
| 근접 DPS (일격·중타) | 야마구치 코타로(일본·사무라이) |
| 원거리 DPS (단일 표적) | 박돌쇠(조선·활쟁이) |
| 원거리 DPS (기동·광역) | 사사키 료조(일본·야부사) |
| 힐러/지원 | 정해은(조선·의원) |
| 캐스터/디버퍼 | 왕청림(명·도사) |
| 버스트/암살 | 후지와라 시노(일본·닌자) |

## 조선 (3명)

### 1. 박돌쇠 — 활쟁이 (궁수)

- **출신**: 한양 변두리, 군호(軍戶) 출신
- **배경**: 중종 때 변방 군에 끌려갔다가 활재주 하나로 살아 돌아온 평민. 양반의 멸시에 지쳐 일정한 주인을 두지 않음. 술과 도박이 약점
- **고유 패시브** — **숨죽임**: 같은 칸 또는 인접 칸에 아군이 없을 때 명중 +20%, 치명 +10%
- **고용 조건**: 평판 0 이상, 보수금 50냥. 한양 「용병 회관」에서 즉시 영입 가능
- **개성**: 처음부터 합류 가능한 입문 캐릭터. 후방 안정적 딜
- **외형 가이드**: 거친 무명옷, 어깨에 활통, 머리에 패랭이

### 2. 김덕림 — 무관 (중장 보병)

- **출신**: 평양 무관 가문 차남, 무과 급제 후 좌천
- **배경**: 정통 무관 출신이나 상관과 충돌해 옷을 벗었음. 자존심 높고 규율 중시. 무리한 의뢰는 거절
- **고유 패시브** — **호위**: 인접 아군이 받는 피해를 30% 대신 받음
- **고용 조건**: 평판 30 이상, 보수금 200냥, 의뢰 「청송 산적단 소탕」 완료
- **개성**: 핵심 방어 탱커. 후방 딜러를 보호하는 기둥
- **외형 가이드**: 두정갑, 환도 + 작은 방패, 단단한 표정의 중년

### 3. 정해은 — 의원 (치유사)

- **출신**: 개성, 중인 의가 출신
- **배경**: 의술을 익혔으나 누이의 죽음 이후 약초학과 함께 주술적 치유를 연구. 요괴 사냥에 합류한 것도 누이의 죽음과 관련된 사건의 단서를 찾기 위해서
- **고유 패시브** — **약초학**: 의뢰 종료 후 부상자 회복 기간 -50%
- **고용 조건**: 평판 20 이상, 보수금 150냥, 의뢰 「개성 약재상의 의뢰」 완료
- **개성**: 유일한 회복원. 그녀가 다운되면 전투가 급격히 불리해짐
- **외형 가이드**: 도포 + 약통, 머리에 동곳, 눈매가 침착함

## 명 (3명)

### 4. 왕청림 (王清林) — 도사 (주술사)

- **출신**: 사천성 도교 산문, 파문된 도사
- **배경**: 산문에서 금지된 술법을 익혔다가 추방됨. 조선으로 흘러들어 요괴 사냥에 종사. 동료의 죽음을 가벼이 여기지 않지만 표현이 차가움
- **고유 패시브** — **부적**: 전투 시작 시 무작위 아군 1명에게 「피해 30% 감소」 부적 부여 (전투 종료까지 지속)
- **고용 조건**: 평판 40 이상, 보수금 300냥, 의뢰 「제물포 항구의 산 자를 부르는 자」 완료
- **개성**: 광역 디버프와 방어 부적. 다음 행동을 봉인하는 「정지부」가 보스전의 키
- **외형 가이드**: 청색 도복, 손에 부적 다발, 길게 자란 수염

### 5. 진소걸 (陈小杰) — 권법가 (타격가)

- **출신**: 산둥 표국, 어릴 적부터 표사 견습으로 자란 청년
- **배경**: 무기 없이 맨손과 짧은 막대로 싸우는 외문권 수련자. 명랑하고 호기심 많음. 조선 음식을 좋아함
- **고유 패시브** — **연환각**: 같은 적을 3회 연속 타격 시 다음 타격이 자동 치명타
- **고용 조건**: 평판 10 이상, 보수금 100냥. 한양 「용병 회관」에서 즉시 영입 가능
- **개성**: 초기 영입 가능한 근접 DPS. 한 표적을 집요하게 두들기는 캐릭터
- **외형 가이드**: 짧은 한복풍 옷, 손목에 단단한 권갑, 미소 띤 청년

### 6. 호중산 (胡中山) — 표사 (반격 탱커)

- **출신**: 남직예성, 대형 표국의 호위 표사
- **배경**: 상단 호위로 조선까지 따라왔다가 표국 본거지가 망해 발이 묶임. 대도(大刀)를 다루며 카운터 위주의 전투 스타일. 말수 적음
- **고유 패시브** — **순간 받아치기**: 인접 적에게 공격받을 때 30% 확률로 즉시 반격 (자동 발동)
- **고용 조건**: 평판 35 이상, 보수금 250냥, 의뢰 「북방 상단 호위」 완료
- **개성**: 받는 만큼 돌려주는 공격형 탱커. 김덕림과 역할이 다름 (덕림 = 차단, 중산 = 반격)
- **외형 가이드**: 가죽 갑옷 + 대도, 어깨에 표국 깃발 흔적, 굵은 흉터

## 일본 (3명)

### 7. 야마구치 코타로 (山口 小太郎) — 사무라이 (검사)

- **출신**: 큐슈, 몰락한 영주 가문의 차남
- **배경**: 주군이 전사한 뒤 떠돌이가 됨(낭인). 조선으로 흘러들어 의뢰를 받음. 명예에 집착하고 약한 자를 베지 않음. 임진왜란 발발 시 가장 큰 갈등을 겪는 캐릭터
- **고유 패시브** — **일도양단**: 적의 HP가 25% 이하일 때 명중 시 즉사 (보스급 제외)
- **고용 조건**: 평판 25 이상, 보수금 200냥, 의뢰 「동래 포구의 떠돌이 검객」 완료 (이 의뢰가 그의 영입 이벤트)
- **개성**: 한 방의 강한 캐릭터. 처치 보조 역할에 강함
- **외형 가이드**: 색바랜 카타기누 + 카타나 + 와키자시, 묶은 머리

### 8. 후지와라 시노 (藤原 忍) — 닌자 (암살자)

- **출신**: 이가국, 닌자 가문의 사생아 출신 여성
- **배경**: 가문에서 환영받지 못해 자의로 떠남. 정보 수집과 암살 전문. 동료와도 거리를 두지만 한 번 신뢰하면 의리 깊음. 여성 캐릭터
- **고유 패시브** — **그림자 이동**: 전투 시작 시 1회 한정, 어떤 칸으로든 순간이동 가능 (자동 사용 안 함, 수동 발동)
- **고용 조건**: 평판 50 이상, 보수금 400냥, 의뢰 「청송 호식인」 완료 후 한양 야시장 이벤트 발생
- **개성**: 후위 캐스터/힐러 암살용. 첫 턴 처치 가능
- **외형 가이드**: 어두운 색 짧은 옷, 얼굴 가리개, 양손에 단검 + 표창

### 9. 사사키 료조 (佐々木 良三) — 야부사 (기마/원호 궁수)

- **출신**: 시코쿠, 무가 출신 야부사메(流鏑馬) 명인
- **배경**: 활쏘는 의식 명인이었으나 의식만으로 만족 못해 실전을 찾아 떠남. 박돌쇠와는 같은 활잡이로서 친밀하지만 출신 차이로 갈등도 있음
- **고유 패시브** — **풍전사**: 이동 후 즉시 사격 가능 (이동 비용 후 별도 행동 비용 없음, 1회/전투)
- **고용 조건**: 평판 30 이상, 보수금 250냥, 의뢰 「임진강변의 흑궁수」 완료
- **개성**: 기동 사수. 박돌쇠가 정 위치 저격이라면 료조는 치고 빠지는 광역 사수
- **외형 가이드**: 가벼운 하카마, 긴 화궁, 등에 짧은 칼

## 영입 진행 흐름

```
[게임 시작]
   │
   ├─ 박돌쇠, 진소걸 (즉시 영입 가능, 입문)
   │
   ├─ [평판 20+] 정해은 → 의원 회복원 확보
   │
   ├─ [평판 25+] 코타로 (낭인 의뢰 진행)
   │
   ├─ [평판 30+] 료조, 김덕림 → 본격 전투 라인업
   │
   ├─ [평판 35+] 호중산 → 탱커 2번째
   │
   ├─ [평판 40+] 왕청림 → 캐스터
   │
   └─ [평판 50+] 시노 → 마지막 영입 (히든)
```

## 임진왜란 발발 시 변화

- **코타로**: 「이방인이지만 너희가 내 동료다」 분기 이벤트. 플레이어 선택으로 잔류/이탈
- **시노**: 정보 수집 능력 강화 이벤트 (특수 의뢰 해금)
- **료조**: 단편 회상 이벤트. 잔류 확정
- **3국 갈등 이벤트**: 한양 숙소에서 일본인 용병 vs 조선인 용병 충돌 → 플레이어 중재 선택
- 일본 용병이 전투 패배 시 일반 NPC 적대 반응 강화 (도시에서 야유 등)
