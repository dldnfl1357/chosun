# 스킬 시스템 + 캐릭터 스킬 키트 (모바일)

**모바일 컨텍스트**: 한 전투에 5명 출전 (주인공 + 용병 4). 8×6 그리드. 자동 스킬은 감빗으로 자동 발동, 수동 스킬은 화면 하단 스킬 바에서 캐릭터 선택 후 탭하여 시전.

**모바일 UI 흐름**:
- 화면 하단: 출전 5명의 초상화 + HP 바
- 캐릭터 탭 시: 해당 캐릭터의 수동 스킬 버튼 2개 부각 (CD 표시)
- 스킬 탭 → 대상 셀 탭으로 시전 (시전 가능 셀 하이라이트)
- 한 번 더 같은 캐릭터 탭 시: 스킬 바 닫힘
- 일시정지 버튼은 우측 상단 (큰 버튼)
- 속도 1x/2x/4x 토글은 일시정지 옆

## 1. 스킬 시스템 규칙

### 두 종류의 스킬
- **자동(Auto)**: 감빗 규칙에 의해 자동 발동. 짧고 자주 쓰는 스킬 위주
- **수동(Manual)**: 플레이어가 일시정지하고 직접 지정. 강력하고 자주 못 씀

### 자원 시스템 (단순화)
- MVP는 **자원 없음, 쿨다운만 사용** (스태미나/마나 없음)
- HP만 자원처럼 취급. 일부 스킬은 자기 HP 소모로 강화 가능
- 「ULT」 표기는 1회/전투 한정 (전투 종료 시 자동 리셋)

### 쿨다운 단위
틱(tick) 기준. 1틱 = 100ms.

| 등급 | 쿨다운 | 의미 |
|------|--------|------|
| 즉시기 | 5–15틱 | 평타 등 빠른 행동 |
| 단주기 | 30–80틱 | 자주 쓰는 자동 스킬 |
| 중주기 | 150–400틱 | 수동 스킬 |
| ULT | 1회/전투 | 결정타급 수동 |

### 사거리 표기
- **Self**: 자신만
- **Melee**: 인접 1칸
- **Tile(N)**: 직선 N칸 (예: Tile(5) = 5칸 사거리 단일 표적)
- **Line(N)**: 직선 N칸 광역 (관통)
- **Cone(N, deg)**: 부채꼴
- **AoE(r)**: 반경 r칸 원형 (대상 칸 중심)

### 스킬 스키마 (구현용)
```csharp
class SkillDef {
  string Name;
  SkillType Type;         // Auto | Manual
  RangePattern Range;
  TargetKind Target;      // Enemy | Ally | Self | Tile
  int CastTimeTicks;
  int CooldownTicks;      // -1 = OncePerBattle
  Effect[] Effects;
  bool GambitCompatible;  // Manual은 항상 false
  string Description;
}
```

### 감빗 호환성
- 모든 Auto 스킬: 감빗 룰 슬롯에 사용 가능
- 모든 Manual 스킬: 감빗 슬롯 불가 (오직 수동 발동)
- 「반응기」 패시브성 자동 스킬은 항상 발동 (감빗과 별개)

## 2. 공통 스탯

전 캐릭터 공통 스탯 (단순화):

| 스탯 | 의미 | 범위 |
|------|------|------|
| HP | 체력 | 50–200 |
| 명중(Acc) | 공격 적중률 보정 | -50 ~ +50 |
| 회피(Eva) | 공격 회피 보정 | -50 ~ +50 |
| 방어(Def) | 받는 피해 감소율 | 0–60% |
| 치명(Crit) | 치명타 확률 | 0–50% |
| 행동속도(Spd) | 모든 스킬 캐스트타임에 곱연산 | 0.5–1.5 |

전투 외 스탯 (지속):
- **사기**: 0–100. 낮으면 명중·회피 감소. 회복은 휴식 또는 정해은 패시브
- **충성도**: 0–100. 낮으면 이탈

## 4. 시너지 매트릭스 (대표 콤보)

| 콤보 | 작동 | 사용처 |
|------|------|--------|
| **김덕림 도발 → 호중산 표사의 자세** | 적이 김덕림 노림 → 호중산 자동 반격으로 정리 | 잡몹 떼 |
| **왕청림 정지부 → 코타로 발도** | 적 행동 봉인 → 첫 공격 보너스 살림 | 중급 단일 표적 |
| **시노 감지 → 박돌쇠 일격필살** | 구미호 본체 식별 → 한방 처치 | 구미호 |
| **왕청림 귀살부 → 정해은 축귀침** | 영체 즉사 + 부적 정리 | 저승사자, 손각시 |
| **진소걸 권타 ×3 → 권타 → (자동 치명)** | 패시브 활용, 단일 표적 폭딜 | 강급 단일 표적 |
| **료조 측면 사격 ⇄ 코타로 추격** | 코타로 이동 → 료조가 「행동 중」 보너스 → 후위 처치 | 캐스터 적 |
| **호중산 부동심 → 정해은 회복의 향** | 무적 동안 광역 회복으로 회복 시간 확보 | 위기 회복 |
| **시노 그림자 이동 → 암살** | 첫 턴 후방 캐스터 즉시 제거 | 그림자 손각시 그룹 |

## 5. 감빗 룰 예시 (각 캐릭터 기본 세팅)

전투 시작 시 자동 적용. 플레이어가 한양에서 수정 가능.

### 박돌쇠 기본 감빗
1. [자신 인접 1칸에 적 있음] → 후퇴사격 (CD 가능 시)
2. [자신 「정조준」 미보유, CD 가능] → 정조준
3. [기본] → 평타 사격, 가장 약한 표적

### 정해은 기본 감빗
1. [HP <30% 아군 있음] → 치유의 손
2. [디버프 보유 아군 있음] → 해독
3. [기본] → 침술, 가장 가까운 적

### 김덕림 기본 감빗
1. [자신 HP <60%, 「굳건한 자세」 CD 가능] → 굳건한 자세
2. [적 3명 이상 자신 주변 3칸] → 도발
3. [기본] → 환도 일격, 가장 가까운 적

(이 패턴으로 9명 모두 기본 감빗 사전 정의. 별도 데이터로 ScriptableObject에 보관)

## 6. 디자인 원칙 (재확인)

1. **자동 3개로 90% 전투 굴러감**. 플레이어는 수동 스킬 사용 타이밍에만 집중
2. **수동 ULT는 결정타용**. 1회/전투이므로 절약 본능
3. **시너지가 영입 동기**. 정지부+발도, 감지+일격필살 같은 콤보가 영입 의의
4. **MVP 이후 확장 여지**: 각 캐릭터에 6번째·7번째 스킬 (트레이닝/렙업)로 추가
